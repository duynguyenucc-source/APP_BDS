# APP_BDS

Workspace tong hop cho cac du an va tai lieu lien quan.

## Cau truc chinh

- `docs/`
  - Tai lieu va spec san pham.
  - Hien tai co spec quan ly cam co bat dong san tai Ben Tre.

- `Ai-wedsite/`
  - Du an Next.js tach rieng.
  - Repo GitHub: `https://github.com/duynguyenucc-source/ai-wedsite`

- `gitpull/so-ban-hang-ai/`
  - Du an Vite + React tach rieng.
  - Repo GitHub: `https://github.com/duynguyenucc-source/so-ban-hang-ai`
